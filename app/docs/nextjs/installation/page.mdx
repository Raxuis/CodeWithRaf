    # How to install Next.js

## Package managers
To create a new Next.js project, you can use one of the following package managers:
- [npm](https://npmjs.com)
- [yarn](https://yarnpkg.com)
- [pnpm](https://pnpm.io)
- [bun](https://bun.sh)

Then add the following command to your terminal to create a new Next.js project:

<details>
    <summary>npm</summary>

    ```shell
    npx create-next-app@latest
    ```
</details>
<details>
    <summary>yarn</summary>
    ```shell
    yarn create next-app@latest
    ```
</details>
<details>
    <summary>pnpm</summary>

    ```shell
    pnpm create next-app@latest
    ```
</details>
<details>
    <summary>bun</summary>

    ```shell
    bun create next-app@latest
    ```
</details>
