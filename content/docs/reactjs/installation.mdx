## React.js Installation
#### Prerequisites:
- Node.js (LTS)
- npm, yarn, or pnpm

#### Steps:
1. Install React:
```bash
npx create-react-app my-react-app
cd my-react-app
npm run dev
   ```
2. Or using yarn:
```bash
yarn create react-app my-react-app
cd my-react-app
yarn dev
   ```
3. Or using pnpm:
```bash
pnpx create-react-app my-react-app
cd my-react-app
pnpm dev
   ```
4. Visit `http://localhost:3000`
