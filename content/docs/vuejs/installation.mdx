### Vue.js Installation
#### Prerequisites:
- Node.js (LTS)
- npm, yarn, or pnpm

#### Steps:
1. Install Vue CLI:
```bash
npm install -g @vue/cli
   ```
2. Create Vue project:
```bash showLineNumbers
vue create my-vue-app
cd my-vue-app
npm run dev
   ```
3. Or using Vite:
```bash showLineNumbers
npm create vite@latest my-vue-app --template vue
cd my-vue-app
npm install
npm run dev
   ```
4. Visit `http://localhost:5173` (Vite) or `http://localhost:8080` (CLI)
