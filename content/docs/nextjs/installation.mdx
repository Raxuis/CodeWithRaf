## Next.js Installation
#### Prerequisites:
- Node.js (LTS)
- npm, yarn, or pnpm

#### Steps:
1. Install Next.js using npm:
```bash
npx create-next-app@latest my-next-app
cd my-next-app
npm run dev
   ```
2. Or using yarn:
```bash
yarn create next-app@latest my-next-app
cd my-next-app
yarn dev
   ```
3. Or using pnpm:
```bash
pnpx create-next-app@latest my-next-app
cd my-next-app
pnpm dev
   ```
4. Visit `http://localhost:3000`
